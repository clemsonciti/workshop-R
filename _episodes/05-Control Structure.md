## If-Else
Syntax:

**if**
```r
if (condition){
  #do task
}
```

**if...else**
```r
if (condition1){
  #do task 1
} else {
  #do the rest
}
```

**if...elseif...else**
```r
if (condition1){
  #do task 1
} elseif (condition2) {
  #do task 2
} else {
  #do the rest
}
```
